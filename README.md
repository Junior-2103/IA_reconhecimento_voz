# Whisper para reconhecimento de voz
Isso é um projeto de reconhecimento de voz para anotações onde
a biblioteca **Pytubefix** pega a URL do video do youtube e 
instala o video na pasta "video" e depois pega o nome com a
biblioteca **os** e transcreve com a biblioteca **whisper** da
**OpenAi** e logo em seguida faz algumas anotações sobre o video
transforma em um arquivo MarkDown para a melhor visualização.
